# Connect Four Game

A simple Connect Four game implemented in Java with a graphical user interface (GUI) built using Swing.

## Overview

This project is a Java-based implementation of the classic Connect Four game. The graphical user interface (GUI) allows two players to play the game interactively. The game is implemented with the use of Swing components to create the visual elements and handle user interactions.

## Features

- Two-player game mode.
- Intuitive GUI built with Swing.
- Highlights the winning sequence.
- Customizable board appearance with thick borders and a checkered border pattern.
- Game status display to indicate the current player.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) installed.
- A Java IDE like IntelliJ IDEA, Eclipse, or BlueJ.

### Installation

1. Clone the repository:
    git clone https://github.com/your-username/ConnectFour.git

2. Navigate to the project directory:
    cd ConnectFour Java Game

### Running the Game

#### Using Command Line

3. Compile the Java files:
    javac ConnectFourGUI.java ConnectFour.java

4. Run the game:
    java ConnectFourGUI

#### Using BlueJ

1. Open BlueJ.
2. Create a new project or open an existing one where you want to add the Connect Four files.
3. Add all the `.java` files (`ConnectFourGUI.java`, `ConnectFour.java`, etc.) to your BlueJ project.
4. Compile the files within BlueJ.
5. Right-click on the `ConnectFourGUI` class and select "void main(String[] args)" to run the game.

## Usage

- Launch the game and the GUI will display the Connect Four board.
- Players take turns clicking on the columns to drop their pieces.
- The game will indicate when a player wins or if there's a tie.
